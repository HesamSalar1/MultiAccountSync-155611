
# سیستم بات تلگرام سه‌گانه

این پروژه شامل 3 بات تلگرام مجزا است که بر روی اکانت‌های مختلف اجرا می‌شوند.

## ویژگی‌ها

- 🤖 3 بات مجزا با اکانت‌های مختلف
- 🔥 مدیریت فحش‌ها و پاسخ خودکار به دشمنان
- 😊 مدیریت دوستان و کلمات دوستانه
- 📊 آمارگیری کامل
- 📤 ارسال همگانی
- 🗄️ ذخیره‌سازی در SQLite

## دستورات

### مدیریت فحش‌ها
- `/addfosh [متن]` - اضافه کردن فحش
- `/delfosh [متن]` - حذف فحش
- `/listfosh` - نمایش لیست فحش‌ها

### مدیریت دشمنان
- `/setenemy` - اضافه کردن دشمن (ریپلای)
- `/delenemy` - حذف دشمن (ریپلای)

### مدیریت دوستان
- `/setfriend` - اضافه کردن دوست (ریپلای)
- `/delfriend` - حذف دوست (ریپلای)

### مدیریت کلمات دوستانه
- `/addword [متن]` - اضافه کردن کلمه
- `/delword [متن]` - حذف کلمه

### تنظیمات
- `/autoreply` - فعال/غیرفعال پاسخگویی
- `/stats` - نمایش آمار
- `/broadcast [پیام]` - ارسال همگانی
- `/help` - راهنما

## اجرا

برای اجرای همه بات‌ها دکمه Run را بزنید یا:

```bash
# اجرای بات 1
cd bots/bot1 && python3 main.py

# اجرای بات 2
cd bots/bot2 && python3 main.py

# اجرای بات 3
cd bots/bot3 && python3 main.py
```

## تنظیمات

هر بات در فایل `main.py` خود تنظیمات زیر را دارد:
- `api_id` - شناسه API
- `api_hash` - کلید API
- `admin_id` - شناسه ادمین

## دیتابیس

هر بات دیتابیس SQLite مجزای خود را دارد:
- `bot1_data.db`
- `bot2_data.db`
- `bot3_data.db`

## لاگ‌ها

لاگ‌های هر بات در فایل‌های جداگانه ذخیره می‌شود:
- `bot1.log`
- `bot2.log`
- `bot3.log`
